YouTube to MP3 ⚰️

Convert YouTube videos to MP3 audio files (abandoned project)

How it Would Work (In Theory)

  • Paste a YouTube URL in the field above
  • Click "Convert to MP3" to start the conversion
  • Watch as multiple APIs immediately fail
  • Contemplate the futility of fighting YouTube's anti-scraping measures
  • Visit yt1s.com directly instead

Why This Project Failed

  • YouTube hates fun: They actively block all automated download attempts
  • API graveyard: Every service we tried is either dead, rate-limited, or shuttered by YouTube's legal team
  • CORS demons: Browsers refuse to talk to external services for "security reasons"
  • The great API collapse of 2024: Services like cobalt.tools shut down literally weeks before we tried them
  • We're not smart enough: We needed a backend server, but we wanted to keep it simple. Narrator: We did not keep it simple.

What We Learned

  • YouTube-to-MP3 conversion is a fight you cannot win with client-side code alone
  • Every free API eventually dies fighting YouTube's legal department
  • The WASM approach would have just made us sad faster, but bigger
  • Sometimes the simplest solution is admitting defeat and accepting limitations
  • This took way longer than it should have. We're not mad, we're just disappointed (mostly in ourselves)
